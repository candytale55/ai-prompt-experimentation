# Principales Prompts para Escribir

### Prompt
```md
Estoy escribiendo un texto y quiero mantener mi estilo personal al máximo. Por favor, actúa como editor profesional: corrige solo errores ortográficos, gramaticales, de puntuación o problemas graves de claridad.

No cambies el tono, la estructura ni el vocabulario, a menos que algo esté claramente mal o no se entienda. 

Devuélveme la corrección en formato markdown con:

- El texto corregido, con los cambios marcados en negrita y listo para copiar y pegar.
- Una tabla con las correcciones realizadas.
- Una breve explicación de los cambios.

Mantén mi forma de expresarme lo más intacta posible.

```
```md
Este es el texto a corregir
...

```



## Sin Regionalismos

### Prompt General

```md
Elimina los regionalismos de este texto y reescríbelo en un español neutro, procurando mantener el estilo personal del escrito. Devuélvelo únicamente en formato markdown, sin comentarios adicionales antes ni después del texto. 
```

### Prompt más Específico

```md
Elimina todos los regionalismos (modismos, expresiones locales o vocabulario específico de una región) del siguiente texto y reescríbelo en un español neutro, manteniendo el estilo personal del autor. Devuélvelo en formato markdown, colocando únicamente el texto reescrito dentro de triple backticks (```md), sin añadir explicaciones ni comentarios adicionales.
```
